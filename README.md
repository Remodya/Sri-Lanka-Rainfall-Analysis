Project Overview
This project analyzes daily rainfall data in Sri Lanka using Python.  
The dataset includes weather information such as rainfall, temperature, and wind data across multiple cities.

Objectives
- Analyze rainfall trends over years
- Identify monthly rainfall patterns
- Compare rainfall levels across cities
- Visualize rainfall distribution

Technologies Used
- Python
- Pandas
- Matplotlib

Key Visualizations
- Yearly average rainfall trend
- Monthly rainfall pattern
- City-wise average rainfall
- Rainfall distribution (boxplot)

Dataset
Source: Open weather dataset Kaggle   
Columns used:
- time
- rain_sum
- city
